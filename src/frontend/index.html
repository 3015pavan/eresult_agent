Provider Guide

        

          

            Gmail
            Google Workspace EDU, personal Gmail — OAuth2 via Connect Gmail button
          

          

            IMAP
            Any college/university mail server (Zimbra, Dovecot, cPanel), Outlook IMAP, Yahoo IMAP — uses TLS port 993
          

          

            Microsoft 365
            University Microsoft 365 EDU, corporate Exchange Online — Azure AD app credentials
          

          

            Webhook
            Mailgun Inbound Routes, SendGrid Inbound Parse, Postmark, custom MTA forward — POSTs raw MIME to /webhooks/email
